About Our Contact Book 📖

This project is designed to help users securely manage and organize their personal or professional contacts in the cloud ☁️.

With features like adding, updating, deleting, and searching contacts, it provides an efficient user experience with a beautiful dashboard UI.

  • 🔒 Secure Login System
  • ➕ Add / ✏️ Update / 🗑️ Delete Contacts
  • 📤 Upload Profile Pictures
  • 🔍 Live Search & Pagination
  • 📧 JavaMail Notifications (Optional)

Made with ❤️ using Spring Boot, Thymeleaf, MySQL & Bootstrap
